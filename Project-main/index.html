Speculation

The purpose of my project is to allow users to express creativity through collaging. The tool should let users create their own unique composition based on a set assortment of images and icons, as well as a large canvas space. The values that guide my project are accessibility and creative expression, which won’t require prior drawing skills. My tool will lack the requirement for users to draw their own image and allow them to freely experiment with composition. The constraints that I will put in place are the lack of a pen tool to force users away from drawing to focus on composition, and utilising what is given within the tool.

As per the Expression and Information design experiment, users will have control over where they decide to place each image and how they would like to arrange and transform each image (scaling, rotating, image layering, multiple of the same image). Audio and visual feedback may be included to provide a more obvious response to user interaction. For example, sound queues when an image is selected, picked up or dropped, and the darkening of buttons when they have been clicked. The tool will have the inclusion of buttons and sliders to perform different functions, such as adjusting the layering of images, rotating, scaling, as well as an undo and redo button.

Research - Provided Tools

I chose Concrete Nest by Rianna Suen & Struan Fraser as my first provided tool to research. The concept of the word collage is what I found most enjoyable, and I based my initial project idea on the concept of a tool that provides users with existing sprites, which they can arrange in the “nest” (the allocated canvas space that I also want to implement in my project).

The second provided research tool that I researched was the tile-flipping toy by Ash K. I enjoyed the different buttons to transform and customise the outcome, and also aim to implement those in my tool. However, the main component that stuck out to me was the “What?” button, which provided users with guidance on what each button did.

Research - Independently Researched Tools

Bird Bakery by Sebastian Scaini & Del Nordlund is a character-building tool that provides users with different options for each part of the bird and allows for different combinations of parts. The end goal of the tool is for users to generate their own unique bird based on the set options of the tool. I found that despite the seemingly closed constraints and simple concept, the comments were filled with many unique birds, with users expressing their own creativity and style in spite of not directly designing each part themselves. The tool is also highly accessible as it exclusively requires users to click on the icons at the top, excluding the use of keyboard shortcuts or extended mouse clicks.

Under Construction by Wes is a limited art tool that utilises different coloured layers and a cut-out feature that users can freely outline the shape using their mouse. Each layer is a different colour that creates contrast between each cut-out layer. The tool also includes a small bar at the bottom of the canvas where cutouts created by the user can be dragged and dropped into the canvas. Despite the straightforward concept, I discovered that the constraint of having to cut inside a previous cut area to reach the next layer was quite interesting, especially when attempting to create a more intricate design. Prompting users to be creative and considerate of where they were making cuts.
